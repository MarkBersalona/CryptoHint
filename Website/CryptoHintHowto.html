Instructions

The CryptoHint Main screen has 3 main input fields.

The text field (1) is used for entry of a cryptogram word. When empty it displays a "Enter cryptogram word here" prompt. When the user taps the text field the keyboard will be displayed; hitting the Return key will dismiss the keyboard and allow the entered word to be parsed. Parsing consists of removing any numeric digits and punctuation and converting the word to uppercase. The wordlist will then be searched for words matching the cryptogram word letter pattern.

The letter substitution map (2) displays how letters in the cryptogram are mapped to letters in the solution. Tapping the display leads the user to the substitution letter picker screen. When CryptoHint starts up the map is empty; as the user specifies cryptogram/solution letter pairs and cryptogram word/solution word matches, the substitution map is filled in and used to improve subsequent word searches.

The table lists a history of the most recent cryptogram words (3). Tapping on a given word in the list will repeat a search for that word. This saves the user from retyping the cryptogram word.

Type a single word from a cryptogram in the text field. In this example we'll type 'XJFFXU'. CryptoHint will take up to a few seconds to search for words that match the letter pattern, e.g. a 6-letter word where the first and fifth letters are identical and the third and fourth letters are identical. If there are any letters established in the letter substitution map the search will filter the word matches based on the substitution map. If any words are found the Match screen will appear.

The CryptoHint Match screen displays the list of words matching the cryptogram word while complying with the current letter substitution map. If the user selects one of the words, the letters of that word are added to the letter substitution map.

The selector at the bottom of this screen allows the list to be displayed either alphabetically or by letter frequency; common words are listed higher than normal in the letter frequency view.

You can also select individual letters in the letter substitution map by tapping the map itself. The screen that allows you to make the selections appears - it has two picker wheels and a selection button. The left wheel lists the letters from the cryptogram; the right wheel lists available letters for the solution; tap the selection button when the approrpriate cryptogram letter is lined up with the desired solution letter.

Example of Using CryptoHint

XM CPAM BVBAM PZ AMWGPKWFXFSFRL RDBK LPI BAM PZ LPIA AFQDRW. - IKBRRAFXIRMH

OK, this cryptogram has one obvious pattern: BVBAM could be EVERY. Type 'BVBAM' in the text field, though, and CryptoHint finds 382 matches like EVENT, EVERY, AGAIN and AWAKE, all of which are common enough to be in a cryptogram. CryptoHint does find only one word for AMWGPKWFXFSFRL, RESPONSIBLITY, so let's use that first.

XM CPAM BVBAM PZ AMWGPKWFXFSFRL RDBK LPI BAM PZ LPIA AFQDRW. - IKBRRAFXIRMH
BE .ORE ...RE O. RESPONSIBILITY T..N .O. .RE O. .O.R RI..TS. - .N.TTRIB.TE.

Looks good so far. BVBAM now has some letters filled in, so let CryptoHint do a search on it again. No need to retype BVBAM, if you had tried it earlier it will be in the list of recent cryptogram words so just tap its entry: this time only one word comes up, AWARE.

XM CPAM BVBAM PZ AMWGPKWFXFSFRL RDBK LPI BAM PZ LPIA AFQDRW. - IKBRRAFXIRMH
BE .ORE AWARE O. RESPONSIBILITY T.AN .O. ARE O. .O.R RI..TS. - .NATTRIB.TE.

CryptoHint finds THAN and TUAN for RDBK. THAN is probably the correct choice, TUAN (a Malay form of respectful address for a man, equivalent to sir or mister; OR a carnivorous marsupial from Australia) is an obscure word.

XM CPAM BVBAM PZ AMWGPKWFXFSFRL RDBK LPI BAM PZ LPIA AFQDRW. - IKBRRAFXIRMH
BE .ORE AWARE O. RESPONSIBILITY THAN .O. ARE O. .O.R RI.HTS. - .NATTRIB.TE.

You can probably work out the rest. CPAM is probably MORE, PZ is probably OF...

XM CPAM BVBAM PZ AMWGPKWFXFSFRL RDBK LPI BAM PZ LPIA AFQDRW. - IKBRRAFXIRMH
BE MORE AWARE OF RESPONSIBILITY THAN .O. ARE OF .O.R RI.HTS. - .NATTRIB.TE.

The rest are obvious.

BE MORE AWARE OF RESPONSIBILITY THAN YOU ARE OF YOUR RIGHTS. - UNATTRIBUTED

Hints and Tips

  • Start with the longest words with any repeating letters - the list of matching words should be relatively short and the word will be good for many letters for the letter substitution map, which will narrow the search for subsequent words.
  • If a cryptogram word search finds no matches, verify you've typed the word correctly. If there were no typos, there's probably a bad guess in the letter substitution map, and/or a previously selected word was incorrect.
  • A single-letter cryptogram word is either A or I; very rarely it is O.
  • The wordlist includes some proper names like LINCOLN, SHAKESPEARE and ANONYMOUS, try the author's name if it's included in the cryptogram.
