Letter substitution screens

The CryptoHint letter picker screen displays two picker wheels: letters of the cryptogram and letters of the solution. The user may select letter pairs to be added to the substitution map; as each letter pair is selected, those letters are removed from their respective picker wheels. A letter may not be paired with itself.

The Reset Map toolbar button resets the letter substitution map and the letter picker wheels.

The Edit toolbar button leads to the substitution pairs screen, which allows the user to delete specific substitution pairs.

At the substitution pairs screen the letter pairs are listed alphabetically by cryptogram letter. Individual pairs may be deleted by swiping across a list item to enable its red Delete button, and tap the Delete button to delete the letter pair.
